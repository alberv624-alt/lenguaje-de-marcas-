EXPLICACION WWW,INTERNET Y HTML

---lista definiciones---

WWW

Es un sistema interconectado de páginas web públicas accesibles a través de Internet. La Web no es lo mismo que el Internet: la Web es una de las muchas aplicaciones construidas sobre Internet

INTERNET
El término Internet se utiliza para hacer referencia a una red de computadoras interconectadas entre sí a nivel mundial con el objetivo de hacer común información de acceso público
HTML

Es el fundamento o lenguaje sobre el que se construye toda página web, se utiliza para estructurar el contenido de forma sencilla para que cualquier usuario pueda acceder a la información de forma sencilla y útil.
